Hostvn Admin Optimize

== Description ==

* Disable Wp-admin Dashboard widget
* Disable Plugins update, Theme update, Wordpress Update
* Add Google Recaptcha to login form, register form, forgot password form
* Disable Generator meta tag
* Change login errors message
* Enable/Disable redirect to homepage after login
* Woocommerce optimize
* SMTP settings
* CDN config
* Security settings
* Contact Button
* Show/Hide menu in wp-admin
* Change login link
* And more ...

== Installation ==

1. Go to Plugins › Add New.
2. Search for *Hostvn Admin Optimize*.
3. Look for this plugin, download and activate it.
4. The page will redirect you to the settings. Change your login url there.
5. You can change this option any time you want, just go back to Settings -> Hostvn AO Setting.
